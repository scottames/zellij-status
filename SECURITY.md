# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in thts, please report it responsibly.

**Do not open a public issue.**

Instead, email security concerns to the maintainer via the contact information
on the [GitHub profile](https://github.com/scottames), or use
[GitHub's private vulnerability reporting](https://github.com/scottames/thts/security/advisories/new).

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

You can expect an initial response within 7 days. Fixes for confirmed
vulnerabilities will be prioritized and released as soon as practical.
